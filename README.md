# Wiremock GUI
A user interface application for accessing [Wiremock OSS](https://wiremock.org/) server made in Godot. The app interacts with Wiremock through Wiremock admin API ([Link](https://wiremock.org/docs/standalone/admin-api-reference/)).

## Supported features
1. Managing mock server instances to access. (IN PROGRESS)
2. View list of mappings with filtering support. 
3. Create new mapping or clone from an existing mapping.
4. Edit mapping. (IN PROGRESS)
5. Delete mapping.

## Platforms
This project aims to support PC as a primary platform (web, Windows, Mac, Linux, etc.).

Note: Currently, only web is available on export options.

## Prerequisite
1. Godot Engine v4.3 - https://godotengine.org/download

## Project Setup
Just export this project into Godot editor and you are ready to go!
